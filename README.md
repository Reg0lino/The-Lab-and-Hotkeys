# FL Hotkeys G910

A learning web app for FL Studio hotkeys with an interactive keyboard canvas and searchable shortcut list.

## Overview

This project is designed to help FL Studio users learn keyboard and mouse shortcuts quickly through an interactive experience.

- Interactive keyboard visualization for Logitech G910 key layout
- Custom mouse macro overlay based on Logitech G502 input
- Searchable reference panel for shortcut lookup
- Learning center with flashcards and multiple-choice quizzes
- Version 1, with some features still evolving and occasional bugs expected

## Features

- Visual key highlighting for active shortcuts
- Custom G-key macro support and mouse action mappings
- Context-driven shortcut filtering for FL Studio workflows
- Search panel to quickly locate hotkeys
- Learning center for practicing shortcuts in a quiz-style format

## Notes

- The app is built for desktop browsers
- The current keyboard/mouse reference is based on Logitech G910 and G502 hardware
- This is an early release and may contain minor layout or interaction issues

## Usage

Open `index.html` in a browser to use the app.

Hosted demo: https://reg0lino.github.io/FL-Hotkeys-g910/

## Files

- `index.html` — app structure and UI
- `style.css` — styling and layout
- `app.js` — interactive behavior and application logic
- `config.js` / `data.js` — shortcut data and configuration
